Bu sahifa ComfyUI ichidagi node ko'rinishini browser uchun preview qiladi. Shu yerda prompt, rasm, sync URL va tanlash logikasi qanday ishlashini oldindan ko'rib olasiz.

Local server bilan ochsangiz /catalog/prompts.json ni o'qiydi. Xohlasangiz GitHub raw URL berib remote katalogni ham tekshirishingiz mumkin.

  • 1. Pastdagi fake ComfyUI workspace ichida node ko'rinadi.
  • 2. SYNC bossangiz katalog qayta yuklanadi.
  • 3. PREV va NEXT bilan promptlar orasida yurasiz.
  • 4. Bu preview uchun HTML/CSS/JS yetadi; ComfyUI dagi real backend esa baribir Python bilan ishlaydi.
Tavsiya: repo ildizidan local server oching va /demo/ ga kiring.
